Neural Network Evolution: From Perceptrons to Medical AI
Name - Kiran Kumar Pradhan
Roll Number - 125ID0012

This repository tracks the transition from foundational AI math to advanced deep learning for medical diagnostics.

🚀 Key Features
Scratch Implementation: Ground-up build of a Perceptron and a 2-layer network using NumPy to demonstrate backpropagation and decision boundaries.

Multi-layer models trained on the MNIST dataset to achieve high-accuracy handwritten digit recognition.

Medical AI (Pneumonia Detection): A deep CNN utilizing BatchNormalization and Dropout to classify chest X-ray images.

Inference Pipeline: A functional system to upload raw X-rays and receive diagnostic predictions with confidence scores.

🛠 Tech Stack
Core: Python, NumPy, Matplotlib, Scikit-learn.

Deep Learning: TensorFlow, Keras.

Datasets: MNIST, Kaggle Chest X-Ray (Pneumonia).
