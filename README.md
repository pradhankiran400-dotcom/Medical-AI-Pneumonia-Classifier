This repository tracks the transition from foundational AI math to advanced deep learning for medical diagnostics.<br>

<h1>🚀 Key Features</h1>
Scratch Implementation: Ground-up build of a Perceptron and a 2-layer network using NumPy to demonstrate backpropagation and decision boundaries.<br>

Multi-layer models trained on the MNIST dataset to achieve high-accuracy handwritten digit recognition.<br>

Medical AI (Pneumonia Detection): A deep CNN utilizing BatchNormalization and Dropout to classify chest X-ray images.<br>

Inference Pipeline: A functional system to upload raw X-rays and receive diagnostic predictions with confidence scores.<br>

<h1>🛠 Tech Stack</h1>
Core: Python, NumPy, Matplotlib, Scikit-learn.<br>

Deep Learning: TensorFlow, Keras.<br>

Datasets: MNIST, Kaggle Chest X-Ray (Pneumonia).<br>
